# 게임 컨셉

## 한 줄 요약

수어로 말하는 RPG — 플레이어는 실제 수어 동작을 수행하여 NPC와 대화하고 세계를 구한다.

## 핵심 아이디어

**WordsInSilence**의 세계에서는 마법이 사라지고 "침묵"만이 남았다.
이 세계의 주민들은 수어로 의사소통하며, 플레이어 역시 웹캠 앞에서 수어를 배우고 사용해야 한다.

## 게임플레이 루프

```
1. NPC가 수어로 퀘스트/대화 제시
2. 플레이어가 대응 수어를 수행 (웹캠 인식)
3. 인식 결과에 따라 대화 분기 또는 퀘스트 진행
4. 수어를 배울수록 더 많은 NPC와 상호작용 가능
```

## 핵심 경험 가치

- **배움의 즐거움**: 게임을 하면서 자연스럽게 수어를 접함
- **접근성**: 청각 장애인과 비장애인이 함께 플레이 가능
- **성취감**: 수어 동작 성공 시 즉각적인 피드백

## 수직 슬라이스 목표

Phase 4에서 구현할 최소 데모:
- NPC 1명과의 완전한 수어 대화 시퀀스 1개
- 수어 3~5개 인식 (예: 안녕, 감사, 도움, 예, 아니오)
- 퀘스트 시작→진행→완료 흐름 1회
