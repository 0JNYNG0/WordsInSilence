# 수직 슬라이스 범위 (Phase 4 목표)

## 목표

기능성 게임의 핵심 경험을 검증하는 최소 완전한 데모를 제작한다.

## 포함 범위

### 수어 동작 (3~5개)
- 안녕하세요 (인사)
- 감사합니다
- 도와주세요
- 예 (긍정)
- 아니오 (부정)

### 씬 구성 (1개)
- 마을 광장 배경
- NPC 1명 (노인 캐릭터, 수어를 가르쳐주는 역할)
- 플레이어 카메라 피드 표시 (화면 모서리)

### 게임플레이 흐름
1. 게임 시작 → NPC 등장 → 수어로 인사
2. 플레이어: "안녕하세요" 수어 수행 → 인식 성공
3. NPC: 퀘스트 제시 (수어로 "도와주세요" 요청)
4. 플레이어: "도와주세요" 수어 수행 → 성공/실패 분기
5. 완료: 간단한 결말 화면

### 기술 요구사항
- MediaPipe Hand Landmarker 실시간 실행
- 5개 수어 사전 데이터
- 상태 머신 완전 구현
- Windows Standalone 빌드 배포 가능

## 제외 범위

- 이동/전투 시스템
- 다수 NPC
- 저장/불러오기
- BGM/SFX (최소 효과음만)
- 수어 학습 모드 (게임 루프 외)

## 성공 기준

- 수어 인식 정확도 ≥ 70% (실제 수어 사용자 기준, 20회 테스트)
- 인식 지연 ≤ 2초
- 데모 실행 중 크래시 없음
- 처음 보는 사람이 도움 없이 플레이 완료 가능
